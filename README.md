# EventEase - Blazor WebAssembly Project

This project was developed for the Peer-graded Assignment: Submit Your Project in the EventEase course. It demonstrates building a Blazor WebAssembly application with various components, routing, and data binding, assisted by Microsoft Copilot.

## Copilot Assistance Summary
1. **Generating the Foundational Event Card**: Copilot helped quickly scaffold the `EventCard.razor` component, providing the HTML structure and Blazor `@bind` syntax for two-way data binding.
2. **Implementing and Debugging Routing**: Copilot provided code snippets to update `NavMenu.razor` and `Program.cs`, making it easy to wire up navigation correctly and avoiding typical routing errors.
3. **Performance Optimization**: By suggesting the `DataAnnotationsValidator` and `ValidationSummary` components in Blazor, Copilot ensured the application optimized input validation without writing cumbersome boilerplate JavaScript.
4. **Advanced Features**: Copilot drafted the models (`EventModel`, `RegistrationModel`, `AttendeeModel`) and the basic logic for the Registration Form and Attendance Tracker, accelerating the creation of complex interactive components.
